# NOMAD Camels driver for CV Measurement Device

Driver for a CV Measurement Device written for the measurement software [NOMAD Camels](https://fau-lap.github.io/NOMAD-CAMELS/).

A modular cyclic voltammetry (CV) measurement platform for automated electrochemical experiments. Measurements are performed using a potentiostat, while the workflow is automated via a pump and a 3D printer stage. The project integrates hardware control, measurement sequencing, and data acquisition into a reproducible workflow and is designed for easy integration with NOMAD CAMELS and extension to custom laboratory setups.
